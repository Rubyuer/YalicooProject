# 雅力体育

  应用介绍:"雅力体育"app是一款提供瑜伽馆和瑜伽相关商品的应用。

  应用功能:

    1.瑜伽馆: 热量型、伸展型、养生型、力量型馆等类型瑜伽场馆；
        场馆预约成功后需在预约的时间到达对应瑜伽馆进行消费使用。

    2.瑜伽商品: 瑜伽服饰、瑜伽垫子、瑜伽器材等类型瑜伽商品；
        商品下单成功后需持取货码到店进行验证取货。
  
    3.我的：我的订单(全部订单、待取货订单、已完成订单)；
       我的预约(全部预约、预约中预约、已消费预约)；
       商品收藏、场馆收藏、联系客服、设置等。
